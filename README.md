## How to Use the Form

1. Open the [Live Form]
(https://collegeprooooject.infinityfreeapp.com/index.php?i=1) 
in your browser.
2. Fill in all the required fields (Name, Email, Phone, etc.).
3. Click the **Submit** button.
4. The data will be saved in the backend MySQL database automatically.
5. For testing locally:
   - Make sure your local server (XAMPP/WAMP) is running.
   - Import the database `db.sql`.
   - Update `db_config.php` with your database credentials.
